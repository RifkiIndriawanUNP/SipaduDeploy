Tentang SIPADU

Sistem Identifikasi Kelayakan Infrastruktur Daerah

📐 Metodologi Fuzzy

🏥 Rumah Sakit & Puskesmas

Jumlah Ideal = Jumlah Penduduk / 120.000
Fuzzy = Data Asli / Jumlah Ideal

🏫 SD / SMP / SMU

SD: Penduduk Usia SD / 1.600
SMP/SMU: Penduduk Usia / 4.800

🛣️ Jalan

RDI = Panjang Jalan / Luas Wilayah
Fuzzy = (RDI - RDImin) / (RDImax - RDImin)

🏪 Pasar

PKP = (Jumlah Pasar / Jumlah Penduduk) × 100%

🕌 Tempat Ibadah

Rasio = (Jumlah Tempat Ibadah / Jumlah Pemeluk) × 1000 (per agama)
